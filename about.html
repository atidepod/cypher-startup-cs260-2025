Cypher Encryption Demo

This is a quick demo of what goes on behind the scenes with this messaging software. A plaintext message is encrypted using a One-Time-Pad style algorithem. A new random key is generated for each message, ensuring security. The random key is also encrypted using a RSA Modulus cypher, and is packed with HMAC noise to ensure message integrety. Type something into the example box to see the raw values that make up your messages.
